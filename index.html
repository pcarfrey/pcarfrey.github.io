TypeScript

JavaScript that scales.

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript.

Any browser. Any host. Any OS.

Free, open source, developed and maintained by Microsoft.

TypeScript compiles to plain JavaScript.

TypeScript will work on any browser that supports JavaScript.

Works with existing JavaScript projects and third-party libraries.
